# Examen-N3

A simple HTML+JS+CSS flashcard game to study the French N3 diving exam.

## Contibute

* Open this folder in VS Code or in a Github Codespace.
* After the devcontainer is ready you can start a local development server with the Live Server vscode extension, or with the included `npm run start`.  The application will be available at a local URL, typically `http://127.0.0.1:5500`.
* Playwright test can be rune with `npm test`.
